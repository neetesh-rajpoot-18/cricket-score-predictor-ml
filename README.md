# Score 50 Prediction 🏏📊

This project applies **Machine Learning (Decision Tree Classifier)** to cricket player statistics to predict if a player will score 50 runs.

## 🔹 Features
- Dataset with player stats (Balls Faced, Strike Rate, Experience, Scored_50)
- Data preprocessing using **pandas**
- Model training and testing with **scikit-learn**
- Accuracy evaluation for performance insights

## 🔹 Tech Stack
- Python
- pandas
- scikit-learn
- Jupyter Notebook

## 🔹 How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the notebook: `score_50_or_not.ipynb`

## 🔹 Dataset
`scores_50.csv` — contains player statistics and target labels.
